# Healthcare Appointment System

## Overview

Healthcare Appointment System is a web-based application that enables patients to book appointments with doctors while allowing healthcare providers to manage schedules efficiently.

---

## Business Objective

Improve patient experience by enabling digital appointment booking, automated notifications, and efficient doctor scheduling.

---

## Key Features

- Patient Registration
- Doctor Management
- Appointment Booking
- Appointment Cancellation
- Notifications
- Search Doctors
- Admin Dashboard
